# External Integrations — Public Overview

botBlade integrates with external apps through Android intents, deep links,
OAuth, SAF, and browser/custom tabs. No proprietary SDKs are bundled without
explicit review. All sensitive external sends require user preview and create
audit events.

External integration registry is NOT YET IMPLEMENTED. See the design manual at
docs/design/botblade-security-manual/03-terminal-and-external-integrations.md.
